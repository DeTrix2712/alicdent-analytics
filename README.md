# Сквозная аналитика для стоматологической клиники «АЛИДЕНТ»

## Описание проекта

Проект представляет собой систему сквозной аналитики для стоматологической клиники, объединяющую данные из:

-   **AmoCRM** — лиды, воронка продаж, статусы, UTM-метки
-   **МакДент (ERP)** — пациенты, визиты, услуги, оплаты
-   **Meta Ads (Facebook Ads)** — рекламные расходы и эффективность кампаний

Цель системы — показать полный путь клиента:

**реклама → лид → запись → визит → оплаты → повторные визиты → ROMI / LTV**

---

## Архитектура

Проект реализован в виде ETL-пайплайна на Python:

1. **Extract** — получение данных из API или mock-источников
2. **Transform** — очистка, нормализация, приведение к единой схеме
3. **Match** — склейка CRM ↔ ERP по номеру телефона
4. **Metrics** — расчёт бизнес- и маркетинговых метрик
5. **Load** — запись данных в витрину (CSV / БД / Google Sheets)

---

## Структура проекта

```text
src/
├─ clients/        # Клиенты API (AmoCRM, МакДент, Meta Ads)
├─ etl/            # ETL-логика: extract / transform / match / metrics / load
├─ storage/        # Запись в БД или Google Sheets
├─ utils/          # Общие утилиты (логирование, телефоны, даты)
├─ config.py       # Конфигурация и переменные окружения
├─ main.py         # Точка входа (запуск ETL)
tests/             # Минимальные тесты бизнес-логики
```
